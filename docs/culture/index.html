# 公司愿景：

成为技术与数据驱动的、最值得信赖的物流生态伙伴

# 公司使命：

我们致力于： 1.为员工创造一个值得自豪的工作平台； 2.物流要素、物流过程、物流场景的全面数字化； 3.引领物流数字化运营和管理的变革；

# 公司价值观：

客户第一：以实现客户价值为目标，最终达到公司长久持续的经营与发展 团队协作：同心合力，协同作战，最大限度发挥团队的优势 诚信：诚信为立人之本 敬业：敬业为立公司之本 创新：拥抱变化，勇于创新 共赢：分享成长，共创共赢
